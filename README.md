# Куат Сагатович — Frontend React Developer

React/Vite landing page for a frontend developer portfolio.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Content

Main portfolio content lives in `src/App.jsx`. Update arrays such as `projects`, `stack`, `qualities`, and `contactLinks` to personalize the site.

The hero image is stored at `public/images/frontend-workspace-hero.png`.
